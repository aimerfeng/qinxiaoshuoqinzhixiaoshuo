## 技术决策记录

### 2026-02-21: 前端技术栈选型

**决策**: 使用 Next.js 14 + TypeScript + Tailwind CSS

**原因**:
- Next.js 14 App Router 提供更好的服务端渲染和路由管理
- TypeScript 提供类型安全，减少运行时错误
- Tailwind CSS 提供快速的样式开发和一致的设计系统

**状态管理选择**: Zustand

**原因**:
- 轻量级，API 简洁
- 支持持久化存储
- 与 React 18 兼容性好
