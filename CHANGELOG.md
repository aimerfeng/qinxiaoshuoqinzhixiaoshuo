## [0.5.0] - 2026-02-23

### Phase 1.5: 社区与会员体系（进行中）

#### 会员等级体系
- 会员等级数据模型与贡献度计算服务
- 贡献度记录、会员申请、会员审核 API
- 每日贡献度上限控制
- 前端：等级展示组件、贡献度进度条、贡献度明细页、会员申请页、特权说明页

#### 零芥子代币系统
- 钱包数据模型与交易服务
- 每日领取、打赏（含限制检查）、交易记录、余额查询 API
- 前端：钱包页面、每日领取入口、打赏弹窗、交易记录列表、余额显示

#### 社区活动系统
- 活动数据模型与奖励发放服务
- 创建活动、参与活动、进度追踪、活动审核 API
- 前端：活动中心、活动详情、创建活动表单、活动进度展示、我的活动列表

#### 用户个人中心
- 用户主页数据、动态列表、关注/粉丝列表、收藏列表 API
- 前端：个人主页布局、资料卡片、动态/收藏/关注/粉丝 Tab、阅读统计展示

#### 管理后台（基础）
- 管理员认证与用户管理 API

## [0.3.0] - 2026-02-22

### Phase 1b: 重要功能

#### 锚点引用模块
- 锚点详情、引用列表、创建引用、变更检测、上下文获取 API
- 前端：段落选择引用交互、引用 Card 创建、引用预览、内容更新提示、引用跳转

#### 创作者控制台
- 仪表板数据、作品统计、草稿自动保存、图片上传 API
- 前端：控制台布局导航、仪表板、作品管理列表
- Vditor Markdown 编辑器集成（三模式切换、自定义工具栏）
- 章节预览、定时发布、数据统计图表
- 离线草稿保存（IndexedDB + 防抖自动保存）

#### 搜索模块
- PostgreSQL 全文搜索配置
- 搜索 API（作品、用户、章节）、搜索建议、筛选排序
- 前端：搜索页面、搜索输入框、搜索结果列表、筛选面板、搜索历史

#### 通知模块
- 通知创建、通知列表、标记已读、WebSocket 实时推送、未读计数
- 前端：通知图标未读数、通知下拉面板、通知列表页面、WebSocket 连接管理

### Phase 1c: 补充功能

#### PWA 离线支持
- Service Worker 配置、静态资源缓存、章节内容缓存
- IndexedDB 存储封装、离线阅读、网络恢复同步、缓存自动清理

#### 阅读列表管理
- 阅读列表 CRUD、阅读状态自动更新、更新提醒
- 前端：阅读列表页面、想读/在读/已读分类、更新标记、批量操作

#### 创作者数据分析
- 阅读量趋势、章节完成率、引用排行、读者活跃时段统计
- 前端：数据概览卡片、趋势图表、章节数据表格、引用排行列表

## [0.1.0] - 2026-02-21

### Phase 0: 本地开发环境

#### Docker 开发环境
- Docker Compose 配置（PostgreSQL 15 + Redis 7 + MinIO）
- 开发环境启动脚本、健康检查脚本、数据库初始化脚本
- Prisma 初始迁移与种子数据

### Phase 1a: MVP核心功能

#### 项目初始化
- 前端：Next.js 14 (App Router) + TypeScript + Tailwind CSS + Zustand + React Query + Socket.IO + Motion + react-spring + 全局动画组件
- 后端：NestJS + Prisma ORM + 环境变量管理
- 数据库：PostgreSQL + pgvector + Prisma Schema
- Redis：连接配置、缓存服务、会话存储
- 对象存储：S3 客户端（MinIO）、文件上传、图片处理（Sharp）

#### 用户认证
- 注册/登录 API、JWT Token、邮箱验证、密码重置、登录锁定、设备指纹
- 前端：注册/登录/忘记密码/邮箱验证页面、Auth Store、路由守卫、个人资料编辑

#### 作品管理
- 作品 CRUD（创建/详情/更新/删除/列表分页筛选）
- 章节管理（CRUD、排序、版本历史、版本回滚）
- 段落与锚点处理（段落解析、Anchor_ID 生成、内容 Hash、变更追踪）
- 小说上传（TXT/DOCX 解析、章节自动识别、批量导入）
- 漫画上传（CRUD、图片批量上传、压缩包解析 ZIP/RAR、自动压缩缩略图）

#### 沉浸式阅读器
- 小说阅读器：段落渲染（Markdown→HTML）、设置面板、夜间模式、章节导航、进度条、段落操作菜单、翻页模式（react-spring 物理弹性动画）、进度自动保存、插图 Lightbox（手势缩放/滑动关闭）、热键系统
- 漫画阅读器：滚动模式（Webtoon 长条优化）、翻页模式（RTL 双页）、图片懒加载+预缓存、缩放手势、章节切换
- 弹幕系统：Danmaku 数据模型（绑定 Anchor_ID）、CRUD API + WebSocket 实时推送、danmaku 引擎集成、控制面板（密度/开关/透明度）

#### 广场信息流
- Card CRUD、信息流 API（推荐/关注）、点赞/取消点赞、评论 CRUD、热度计算
- 前端：广场页面、Card 组件、引用预览卡片、无限滚动、下拉刷新、发布弹窗、评论列表、推荐/关注流切换

#### 数据持久化与性能
- 数据库优化（索引、连接池、软删除）
- 缓存策略（热门内容、用户会话、计数器缓存）
- 日志与监控（结构化日志、错误追踪、基础性能监控）
